# Twitter Sentimental-Analysis using ML
Dataset being used : Sentiment 140

I. PreProcessing :
  1. Remove URL
  2. Remove @
  3. Remove Apostrophe
  4. Remove Newline
  5. Replace Emoticons with suitable text
  6. Remove stopwords
  7. Stemming (PoterStemmer)
 
 II. Data Training :
    1.Count Vectorization
    2.LinearSVC - Non Convergent
    3.Gaussian RBF SVC
    4.Polynomial SVC
    
    
  III. Results :
      1.Gaussian RBF
